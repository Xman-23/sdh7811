**고급 함수 

화살표 함수
map 함수 ==> 배열의 각각의 요소에 어떤 연산을 하여 다시 배열을 만들어주는 함수
filter 함수 ==>배열의 요소중에 filter의 조건을 만족하는 새로운 배열을 만들어주는 함수

자바스크립트에서 전통적인 함수 만드는 방법은 다음과 같았습니다.

function 함수이름() {

}

떄로는 이름없는 함수를 만들 수 있어요.

}

또, 최신 자바스크립트(모던 스크립트)에서는
